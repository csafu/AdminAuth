# AdminAuth

#### 项目介绍

AdminAuth是基于Springboot实现的面向后台管理的用户鉴权和权限管理。目的是实现开箱即用快速上手、流程简介，适合初学者快速实现前后端分离的后台用户登录鉴权和用户权限角色的赋予。
